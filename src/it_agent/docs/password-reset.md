# Password Reset Procedures

## Self-Service Password Reset

### Via Company Portal
1. Go to https://password.company.com
2. Click "Forgot Password"
3. Enter your company email address
4. Answer your security questions or use your backup email
5. Create a new password following the password policy

### Password Policy
- Minimum 12 characters
- Must include: uppercase, lowercase, number, and special character
- Cannot reuse the last 10 passwords
- Expires every 90 days
- Account locks after 5 failed attempts

## Account Lockout

### Symptoms
- "Your account has been locked" message
- Unable to log in despite correct password
- Multiple failed login attempts

### Resolution
1. Wait 30 minutes for automatic unlock
2. Or contact IT Support to manually unlock
3. If you suspect unauthorized access, report it immediately

## Multi-Factor Authentication (MFA)

### Setting Up MFA
1. Log into the company portal
2. Go to Security Settings > MFA
3. Download Microsoft Authenticator or Google Authenticator
4. Scan the QR code with your authenticator app
5. Enter the 6-digit code to verify

### Lost MFA Device
1. Contact IT Support immediately
2. Verify your identity with your manager
3. IT will issue a temporary bypass code
4. Re-enroll MFA with your new device
