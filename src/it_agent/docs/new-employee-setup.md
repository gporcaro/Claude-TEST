# New Employee IT Setup

## Day 1 Checklist
- [ ] Receive laptop and peripherals from IT
- [ ] Log in with temporary credentials (check welcome email)
- [ ] Change your password immediately
- [ ] Set up Multi-Factor Authentication (MFA)
- [ ] Connect to Corp-Secure Wi-Fi
- [ ] Install VPN client
- [ ] Set up email in Outlook
- [ ] Join required Slack channels
- [ ] Install required software from Company Portal

## Required Software
All software should be installed from the Company Software Portal: https://portal.internal/software

### Standard Install
- Microsoft 365 (Outlook, Teams, Word, Excel, PowerPoint)
- Slack Desktop
- Cisco AnyConnect VPN
- Google Chrome
- Zoom

### Developer Tools (if applicable)
- VS Code
- Git
- Docker Desktop
- Request access to GitHub organization via IT ticket

## Access Requests
- **Shared drives**: Request through your manager
- **SaaS tools** (Jira, Confluence, etc.): Request via IT ticket
- **Admin access**: Requires manager + director approval
- **Production systems**: Requires security training completion

## Getting Help
- Slack: #it-help channel
- Email: it-support@company.com
- Emergency: Call the IT hotline at x5555
- Self-service: https://portal.internal/help
